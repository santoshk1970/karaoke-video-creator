# 🎬 Video Generation Guide

The Lyric Sync tool can now automatically generate MP4 videos from your timed lyrics and images!

## Quick Start

### Generate Video Automatically

```bash
npm start song.mp3 lyrics.txt --video
```

This will create:
- Images (PNG files)
- Timestamps (JSON file)
- **Video (output.mp4)** ✨

### Standalone Video Generation (NEW!)

If you already have images and timestamps, create video only:

```bash
npm run video song.mp3 ./output
```

This requires:
- `output/timestamps.json` (existing)
- `output/images/*.png` (existing)
- Creates: `output/output.mp4`

### With Custom Quality

```bash
npm start song.mp3 lyrics.txt --video --quality ultra
```

Quality options:
- `low` - Fast encoding, larger file
- `medium` - Balanced (default)
- `high` - Better quality, slower
- `ultra` - Best quality, slowest

## How It Works

1. **Reads** your lyrics file
2. **Aligns** lyrics to audio timestamps
3. **Generates** PNG images for each line
4. **Creates** video using FFmpeg:
   - Each image displays for its timed duration
   - Audio is synced perfectly
   - Output is MP4 (H.264 + AAC)

## Output

```
output/
├── images/
│   ├── lyric_000.png
│   ├── lyric_001.png
│   └── ...
├── timestamps.json
└── output.mp4          ← Your video!
```

## Video Specifications

- **Format**: MP4 (H.264 video, AAC audio)
- **Resolution**: 1920x1080 (Full HD)
- **Frame Rate**: 30 FPS
- **Audio**: 192 kbps AAC
- **Compatibility**: Works on all devices/platforms

## Standalone Video Command

### When to Use

Use `npm run video` when you:
- Already have images and timestamps
- Want to regenerate video with different quality
- Want to create multiple video versions
- Made manual adjustments to timestamps.json

### Usage

```bash
npm run video <audio-file> <output-dir> [options]
```

**Options:**
- `--quality <quality>` - Video quality: low|medium|high|ultra
- `--output <filename>` - Output filename (default: output.mp4)

### Examples

```bash
# Basic usage
npm run video song.mp3 ./output

# Custom quality
npm run video song.mp3 ./output --quality ultra

# Custom output filename
npm run video song.mp3 ./output --output karaoke.mp4

# Both options
npm run video song.mp3 ./output --quality high --output final.mp4
```

### Requirements

The output directory must contain:
1. `timestamps.json` - Generated by main tool
2. `images/` folder - With all lyric PNG files

### Workflow Example

```bash
# Step 1: Generate images and timestamps (no video)
npm start song.mp3 lyrics.txt --output ./output

# Step 2: Review/edit timestamps.json if needed
# (manually adjust timing)

# Step 3: Generate video from edited timestamps
npm run video song.mp3 ./output --quality ultra
```

## Advanced: Manual Video Creation

If you already have images and timestamps, you can create a video manually:

### Method 1: Using FFmpeg Concat

```bash
# Create concat file
cat > concat.txt << EOF
file 'images/lyric_000.png'
duration 3.5
file 'images/lyric_001.png'
duration 4.2
file 'images/lyric_002.png'
duration 3.8
file 'images/lyric_002.png'
EOF

# Generate video
ffmpeg -f concat -safe 0 -i concat.txt -i song.mp3 \
  -c:v libx264 -preset medium -crf 23 \
  -c:a aac -b:a 192k -pix_fmt yuv420p \
  -shortest -y output.mp4
```

### Method 2: From JSON Timestamps

If you have `timestamps.json`, you can write a script to generate the concat file:

```javascript
const fs = require('fs');
const data = JSON.parse(fs.readFileSync('timestamps.json'));

const lines = [];
data.lyrics.forEach((lyric, i) => {
    lines.push(`file '${lyric.imagePath}'`);
    lines.push(`duration ${lyric.duration}`);
});
// Add last image again
lines.push(`file '${data.lyrics[data.lyrics.length-1].imagePath}'`);

fs.writeFileSync('concat.txt', lines.join('\n'));
```

Then run FFmpeg as shown above.

## Troubleshooting

### Video is out of sync

**Solution**: Use the `--vocal` option with a vocal-only track for better timing:

```bash
npm start song.mp3 lyrics.txt --vocal vocals.mp3 --video
```

### Video generation is slow

**Solution**: Use lower quality for faster encoding:

```bash
npm start song.mp3 lyrics.txt --video --quality low
```

### FFmpeg error

**Solution**: Verify FFmpeg installation:

```bash
ffmpeg -version
```

If not installed:

```bash
brew install ffmpeg
```

### Images don't match timing

**Solution**: 
1. Check `timestamps.json` for correct timings
2. Manually adjust if needed
3. Regenerate video with corrected JSON

### Video file is too large

**Solution**: Use higher quality setting (better compression):

```bash
npm start song.mp3 lyrics.txt --video --quality ultra
```

## Tips for Best Results

✅ **Use high-quality audio** - Better audio = better video

✅ **Provide vocal track** - Improves lyric timing accuracy

✅ **Check timestamps first** - Review JSON before generating video

✅ **Use appropriate quality** - `high` is good for most cases

✅ **Test with short clips** - Use `low` quality for quick tests

## Batch Processing

Generate videos for multiple songs:

```bash
#!/bin/bash
for song in *.mp3; do
    lyrics="${song%.mp3}.txt"
    output="./videos/${song%.mp3}"
    npm start "$song" "$lyrics" --video --output "$output"
done
```
