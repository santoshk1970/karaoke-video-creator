╔══════════════════════════════════════════════════════════════════════════════╗
║                            💡 EXAMPLE COMMANDS                                ║
╚══════════════════════════════════════════════════════════════════════════════╝

# Basic usage
npm start song.mp3 lyrics.txt

# With options
npm start song.mp3 lyrics.txt --output ./results --format lrc

# With vocal track (better accuracy)
npm start song.mp3 lyrics.txt --vocal vocals.mp3

# Test the tool
npm run build
npm start example-audio.mp3 example-lyrics.txt
