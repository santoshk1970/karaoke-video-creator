╔══════════════════════════════════════════════════════════════════════════════╗
║                           HOW IT WORKS                                        ║
╚══════════════════════════════════════════════════════════════════════════════╝

1. Reads your lyrics file (supports Hindi, Unicode, etc.)
2. Aligns lyrics to audio timestamps
3. Generates PNG images for each line
4. Creates video using FFmpeg:
   - Each image displays for its timed duration
   - Audio is perfectly synced
   - Output is MP4 (works everywhere)

╔══════════════════════════════════════════════════════════════════════════════╗
║                           TECHNICAL DETAILS                                   ║
╚══════════════════════════════════════════════════════════════════════════════╝

Implementation:
- src/videoGenerator.ts - Main video generation class
- Uses FFmpeg concat demuxer for efficiency
- H.264 video codec (universal compatibility)
- AAC audio codec (high quality)
- Configurable quality presets
- Automatic duration calculation from timestamps

Method:
1. Creates FFmpeg concat file from timestamps
2. Each image gets exact duration from timing data
3. Combines with audio track
4. Encodes to MP4 with selected quality
