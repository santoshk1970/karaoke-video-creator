📖 Help Guide

Learn how to create karaoke videos

🚀 Quick Start

  1. 1. Create Project: Upload audio file and lyrics text file
  2. 2. Time Lyrics: Press SPACE when you hear each line (auto-adjusts -2s)
  3. 3. Generate Videos: Creates images and videos automatically
  4. 4. Play: Watch your karaoke videos!

📝 Detailed Steps

Step 1: Time Lyrics

  • Reload Lyrics: Upload a new lyrics file if you made changes
  • Start Timing Tool: Opens timing interface in new window
  • Edit Timing: Fine-tune timing after generation (with propagate feature)
💡 Tips:
  • The timing tool auto-adjusts -2 seconds for reaction time
  • Auto-Instrumental: Gaps longer than 15 seconds are automatically marked as "Instrumental" sections—no need to tag them manually!

Step 2: Generate Videos

  • Automatically generates lyric images
  • Creates sing-along video (with vocals)
  • Creates karaoke video (if instrumental audio exists)
💡 Tip: Use "Separate Vocals" to create instrumental audio for karaoke

Step 3: Play Videos

  • Karaoke: Instrumental audio only (for singing along)
  • Sing-along: Original audio with vocals (for practice)

🎵 Audio Management

Separate Vocals (AI)

Uses AI to extract instrumental audio from your song (takes 1-2 minutes)

Upload Instrumental

If you already have instrumental audio, upload it directly

⏱️ Timing Editor

Fine-tune timing without re-running the timing tool:

  • Edit Times: Adjust start times by 0.01 second increments
  • Propagate: Check box to shift all subsequent lyrics by the same amount
  • Regenerate: One button regenerates both images and videos
Example: Change lyric 5 from 22.00s to 24.00s with propagate → all lyrics after #5 shift +2 seconds

⌨️ Keyboard Shortcuts

SPACE

Mark lyric timing

Q

Quit timing tool

🔧 Troubleshooting

Audio won't stop playing?

Use the red stop button (🛑) in the header to kill all audio

Timing is off?

Use the "Edit Timing" button to adjust without re-timing everything

Video generation failed?

Check the console output for error messages
